| Error Handling | Limited | Callbacks, exceptions (CommitFailedException) |

- `SyncCommitManager` - Blocking synchronous commits
- `AsyncCommitManager` - Non-blocking asynchronous commits  
- `NoopCommitManager` - No offset management

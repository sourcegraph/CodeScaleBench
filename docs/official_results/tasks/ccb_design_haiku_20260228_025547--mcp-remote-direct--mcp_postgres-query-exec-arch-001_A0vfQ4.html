Query Parsing Pipeline\nThe **parser stage** transforms unstructured SQL text into a structured Abstract Syntax Tree (RawStmt). It employs a two-phase approach:\n- **Lexical Analysis (scan.l)**: A Flex-generated lexer tokenizes the input stream, identifying keywords, identifiers, operators, and literals.\n- **Syntactic Analysis (gram.y)**: A YACC-generated bottom-up parser applies grammar rules to construct a tree of statement nodes.

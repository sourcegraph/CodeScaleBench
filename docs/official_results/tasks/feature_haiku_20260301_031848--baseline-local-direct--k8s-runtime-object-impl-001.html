## Background

The `runtime.Object` interface is the foundational interface for all Kubernetes API types. It is defined in the `k8s.io/apimachinery/pkg/runtime` package and requires two methods:

```go
type Object interface {
    GetObjectKind() schema.ObjectKind
    DeepCopyObject() Object
}
```

Every Kubernetes resource type (Pod, Deployment, Service, etc.) implements this interface by embedding `metav1.TypeMeta` and implementing the autogenerated `DeepCopyObject()` method.

## Repositories

Two repositories are available under `/workspace/`:

- `/workspace/api/` — kubernetes/api (standard API types: core/v1, apps/v1, etc.)
- `/workspace/apimachinery/` — kubernetes/apimachinery (runtime machinery, metav1, schema)

## Task

Find **all struct types** that implement the `runtime.Object` interface across both repositories. Focus on:

1. **Concrete resource types** (Pod, Deployment, Service, ConfigMap, etc.)
2. **Types with TypeMeta embedded** — the standard pattern for runtime.Object implementors
3. **Types with DeepCopyObject() methods** — either explicit or autogenerated

For each implementor, record:
- `repo`: either `kubernetes/api` or `kubernetes/apimachinery`
- `file`: path relative to the repository root (e.g., `core/v1/types.go`)

- Generated code files (`*_generated.go`, `zz_generated.*`)

#!/usr/bin/env python3
import os
import re
import json

def find_struct_with_typemeta(file_path):
    """Find all struct types that have metav1.TypeMeta embedded."""
    results = []
    
    try:
        with open(file_path, 'r') as f:
            content = f.read()
    except:
        return results
    
    # Pattern to find: type Name struct { ... metav1.TypeMeta ... }
    # We need to be careful because Go allows multi-line struct definitions
    
    # Split by "type " keyword to find type definitions
    lines = content.split('\n')
    
    i = 0
    while i < len(lines):
        line = lines[i]
        
        # Look for "type NAME struct {" pattern
        type_match = re.match(r'^\s*type\s+([A-Z]\w*)\s+struct\s*\{', line)
        
        if type_match:
            struct_name = type_match.group(1)
            
            # Skip List types
            if 'List' in struct_name:
                i += 1
                continue
            
            # Now look ahead for metav1.TypeMeta in the next few lines
            has_typemeta = False
            j = i + 1
            brace_count = 1
            
            while j < len(lines) and brace_count > 0:
                struct_line = lines[j]
                
                # Count braces to find the end of struct
                brace_count += struct_line.count('{') - struct_line.count('}')
                
                # Check for metav1.TypeMeta
                if 'metav1.TypeMeta' in struct_line:
                    has_typemeta = True
                
                if brace_count == 0:
                    break
                
                j += 1
            
            if has_typemeta:
                results.append(struct_name)
        
        i += 1
    
    return results

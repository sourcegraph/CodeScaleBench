4. **`lib/base64.c`** — Base64 encoding/decoding: used for HTTP authentication, certificate PEM encoding, and data URI handling.

   - Causes crash when processing encrypted certificates without a password callback

3. **Certificate Verification Union Type Confusion (HIGH)** - `lib/vtls/openssl.c:2210`

   - Can lead to out-of-bounds reads of certificate data

   - The `addr` structure used for IP matching is never explicitly initialized
   - Potential use of uninitialized stack memory

5. **Base64 Decoder Integer Validation (MEDIUM)** - `lib/base64.c:110`
   - Missing validation that decoded length is positive
   - Edge case where rawlen calculation could produce invalid values

### 2. **kubernetes/kubernetes (Event Cache & Distribution)**
Multiple coordinated components in `staging/src/k8s.io/apiserver/pkg/storage/cacher/`:

- **`Cacher`** (`cacher.go`) - Top-level orchestrator that:
  - Creates a Reflector with watchCache as destination
  - Registers `processEvent()` as the eventHandler
  - Distributes events to watchers via `dispatchEvents()`

- **`watchCache`** (`watch_cache.go`) - Sliding window buffer that:
  - Stores recent watch events in a cyclic buffer
  - Implements the `cache.Store` interface
  - Calls the eventHandler when events arrive

- **`listerWatcher`** (`lister_watcher.go`) - Bridge to etcd that:
  - Wraps `storage.Interface`
  - Implements `cache.ListerWatcher`
  - Connects to etcd via `storage.Watch()`

### 3. **kubernetes/client-go (Event Consumer)**
- **File**: `tools/cache/reflector.go`
- **Struct**: `Reflector`
- **Key Method**: `ListAndWatch()` - receives watch events and writes to the Store

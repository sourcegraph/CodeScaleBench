CLASS: GradientNoiseInjector
─────────────────────────────

Purpose:
  Optimizer wrapper that injects calibrated Gaussian noise into gradients
  before the optimization step, implementing "Adding Gradient Noise Improves
  Learning for Very Deep Networks" (Zhu et al., 2018)

Constructor:
  GradientNoiseInjector(optimizer, eta=0.01, gamma=0.55)
  
  Parameters:
    • optimizer: Base optimizer to wrap (torch.optim.Optimizer)
    • eta: Initial noise scale (default: 0.01)
    • gamma: Noise decay rate (default: 0.55)

Key Methods:
  • step(closure=None): Injects noise and delegates to base optimizer
  • zero_grad(set_to_none=False): Zeros gradients via base optimizer
  • state_dict(): Saves state including step count
  • load_state_dict(state_dict): Restores from checkpoint
  • __getattr__(name): Delegates attribute access

Noise Schedule:
  variance = eta / ((1 + step_count) ** gamma)
  std = sqrt(variance)
  noisy_grad = grad + torch.randn_like(grad) * std

4. **nsIContentSecurityPolicy.idl** - Declares the XPCOM interface that exposes CSP functionality to Firefox, including directive enums and violation type constants.

5. **CSPEvalChecker.cpp/h** - Implements `CSPEvalChecker` for handling eval() permission checking in script-src directive, with static methods for both windows and workers.

### Inline Script Evaluation Flow:
- **nsCSPParser** → parses CSP policies
- **nsCSPContext** → evaluates directives (`GetAllowsInline()`)
- **ReportInlineViolation()** → reports violations
- **AsyncReportViolation()** → asynchronous violation reporting to observers

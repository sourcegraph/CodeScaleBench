4. **Storage Configuration** (`pkg/registry/networking/rest/storage_settings.go`):
   - **v1 is the storage version** - objects stored in etcd as v1
   - v1beta1 requests are converted on read/write

5. **API Registration** (`pkg/apis/networking/install/install.go`):

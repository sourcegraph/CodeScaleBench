
Computes the rolling (sliding window) median over an array.

**Parameters:**
- `a` : array_like - Input array
- `window_size` : int - Size of the rolling window
- `axis` : int, optional - Axis along which to compute (default: -1)
- `mode` : {'valid', 'same', 'full'}, optional - Edge handling (default: 'valid')

**Returns:**

## Modes Explained

- **'valid'**: Only compute where window fully overlaps (output size: n - w + 1)
- **'same'**: Keep original size, pad edges (output size: n)
- **'full'**: All positions where window touches array (output size: n + w - 1)

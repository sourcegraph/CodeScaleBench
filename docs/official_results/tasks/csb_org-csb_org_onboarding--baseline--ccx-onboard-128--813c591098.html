- CompileBroker receives compilation requests from the application
- It assigns these requests to CompilerThread instances
- These threads dispatch work to either C1 or C2 compiler implementations through the AbstractCompiler interface
- C1 provides quick compilation for frequently executed code
- C2 performs deeper analysis and optimization on hot code paths

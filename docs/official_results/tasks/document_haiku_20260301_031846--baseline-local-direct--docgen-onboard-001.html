### Required Go Version

Istio's control plane requires **Go 1.25.0 or later**. Check your version:

```bash
go version
```

If you need to install or update Go, visit [golang.org](https://golang.org).

### Environment Setup

The Istio build system uses environment variables to control the build process. Key variables include:

- `GOPATH` - Your Go workspace path (automatically detected)
- `GOPROXY` - Go module proxy (defaults to `https://proxy.golang.org`)
- `GO111MODULE` - Enables Go modules (defaults to `on`)
- `BUILD_WITH_CONTAINER` - Set to `1` to use Docker for builds (optional; defaults to local build)

### Cloning the Repository

Clone the repository with all submodules:

```bash
git clone https://github.com/istio/istio.git
cd istio
```

The repository structure includes:
- `pilot/` - Service discovery and traffic management (Pilot)
- `security/` - Certificate management and security policies (Citadel)
- `pkg/` - Shared packages (XDS, bootstrap, configuration)
- `istioctl/` - Command-line tool
- `tests/` - Integration and unit tests

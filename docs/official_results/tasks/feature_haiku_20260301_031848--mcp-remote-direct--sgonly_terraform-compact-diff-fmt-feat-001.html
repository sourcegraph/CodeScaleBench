## Implementation Summary

### 1. **`internal/command/format/compact.go`** - Main Formatter Implementation

  - Returns compact summary output grouped by action type

  - Handles all plan actions: Create, Update, Delete, DeleteThenCreate, CreateThenDelete, Read, Forget, CreateThenForget

**Output Format:**
```
Create resources:
  [+] resource_type.name
  [+] another_resource.name

Update resources:
  [~] resource_type.name

Delete resources:
  [-] resource_type.name
```

- `actionLabel()` - Maps action types to human-readable labels

- Nil plan and nil changes handling

// Format returns a compact summary of changes in the plan, grouped by action type.
// It displays one line per resource change showing the action and resource address.
func (f *CompactDiffFormatter) Format(plan *plans.Plan, schemas *schemarepo.Schemas) string {
	if plan == nil || plan.Changes == nil {
		return ""
	}

	// Group changes by action type
	changesByAction := make(map[plans.Action][]*plans.ResourceInstanceChangeSrc)

	// Iterate through all resource changes, filtering out deposed objects
	for _, rc := range plan.Changes.Resources {
		// Skip deposed keys (only show current objects)
		if rc.DeposedKey != states.NotDeposed {
			continue
		}

		changesByAction[rc.Action] = append(changesByAction[rc.Action], rc)
	}

	var output strings.Builder

	// Define the order of actions to display
	actionOrder := []plans.Action{
		plans.Create,

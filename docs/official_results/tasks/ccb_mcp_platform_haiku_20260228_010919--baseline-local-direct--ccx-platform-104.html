## Key Findings

The LLVM loop vectorization infrastructure consists of the following components:

1. **LoopVectorizePass** - The main pass interface defined in `llvm/include/llvm/Transforms/Vectorize/LoopVectorize.h` (line 131) as a struct that drives the vectorization process.

4. **LoopVectorizationCostModel** - The cost model class defined in `llvm/lib/Transforms/Vectorize/LoopVectorize.cpp` (line 882) that determines profitable vectorization factors.

5. **VPlan** - The Vectorization Plan data structure defined in `llvm/lib/Transforms/Vectorize/VPlan.h` (line 4491) that represents the vectorized loop body as a hierarchical CFG.

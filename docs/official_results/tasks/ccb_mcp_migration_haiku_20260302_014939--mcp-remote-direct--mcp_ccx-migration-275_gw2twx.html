**StorageVersion Manager:**
- Interface defined in `staging/src/k8s.io/apiserver/pkg/storageversion/manager.go`
- Manages tracking of which versions have been registered and synced
- Uses ResourceInfo struct to track encoding versions and decodable versions

**ResourceEncodingConfig:**
- Interface in `staging/src/k8s.io/apiserver/pkg/server/storage/resource_encoding_config.go`
- Controls which GroupVersion is used for etcd storage through StorageEncodingFor()
- DefaultResourceEncodingConfig maintains a map of GroupResource to OverridingResourceEncoding
- SetResourceEncoding() method allows mapping v1beta1 resources to v1 storage encoding

**Version Registration Pattern:**
- Each storage provider defines separate v1Storage(), v1beta1Storage(), etc. methods
- NewRESTStorage() orchestrates registering all versions in VersionedResourcesStorageMap
- The same underlying storage implementations are shared across API versions
- ResourceEncodingConfig determines the canonical storage version

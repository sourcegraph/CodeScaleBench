## Key Findings

**Apache Kafka's log compaction consists of 13 main Java source files:**

### Core Components:

   - Manages multiple cleaner worker instances

   - Implements "dirtiest first" selection strategy
   - Selects logs with highest cleanableRatio (percentage of cleanable bytes)
   - Considers minCleanableRatio and compaction lag constraints

   - **Segment Rewriting**: `cleanSegments()` and `cleanInto()` methods rewrite log segments

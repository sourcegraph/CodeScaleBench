# Cilium eBPF Map API Reference

## Overview

The Cilium eBPF Map API provides Go wrappers for user-space interaction with kernel eBPF maps. The `Map` type is the primary interface for all map operations, wrapping the lower-level ebpf-go library with Cilium-specific functionality including caching, event streaming, pinning, and metrics.

All eBPF maps are stored in a BPF filesystem (bpffs) and can be persisted between program runs through pinning.

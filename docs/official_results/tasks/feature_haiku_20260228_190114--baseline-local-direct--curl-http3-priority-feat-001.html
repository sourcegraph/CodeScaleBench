# `--h3-priority`

Set the HTTP/3 stream priority according to RFC 9218 Extensible Priorities.

The priority is specified as a string in the format `u=<urgency>[,i]` where:

- `<urgency>` is a number between 0 and 7, where 0 is the highest priority and 7 is the lowest priority (default: 3)
- `,i` is an optional incremental flag (default: false)

Examples:
- `u=0` - Highest priority, non-incremental
- `u=3` - Default priority (non-current), non-incremental
- `u=7` - Lowest priority, non-incremental
- `u=3,i` - Default priority, incremental
- `u=0,i` - Highest priority, incremental

This option only has effect when using HTTP/3.

## Summary

**Five Core Components of the HotSpot JIT Pipeline:**

1. **CompileBroker** (`src/hotspot/share/compiler/compileBroker.hpp`)
   - The central broker managing all compilation requests
   - Maintains separate compile queues for C1 and C2
   - Coordinates compiler thread initialization and management

2. **CompilerThread** (`src/hotspot/share/compiler/compilerThread.hpp`)
   - Dedicated thread type that executes compilation tasks
   - Inherits from JavaThread
   - Manages compilation environment, logging, and task state

3. **Compiler (C1)** (`src/hotspot/share/c1/c1_Compiler.hpp`)
   - First-tier lightweight JIT compiler
   - Inherits from AbstractCompiler
   - Provides quick compilation with minimal optimization overhead

4. **C2Compiler** (`src/hotspot/share/opto/c2compiler.hpp`)
   - Optimizing JIT compiler (second tier)
   - Inherits from AbstractCompiler
   - Implements aggressive optimizations (escape analysis, superword vectorization, etc.)

5. **AbstractCompiler** (`src/hotspot/share/compiler/abstractCompiler.hpp`)
   - Base class for all JIT compilers (C1, C2, JVMCI)
   - Defines common interface for initialization, compilation, and thread management
   - Enables the two-tier adaptive compilation strategy

## Build Prerequisites

### Required Go Version

Istio requires **Go 1.25.0 or later**. Check your current version:

```bash
go version
```

If you need to upgrade Go, visit [golang.org/dl](https://golang.org/dl).

### Required Tools and Dependencies

The main build system uses Make. Ensure you have:
- `make` (GNU Make)
- `git`
- `docker` (optional, if using `BUILD_WITH_CONTAINER=1`)
- Go build tools (normally included with Go installation)

### Cloning the Repository

Clone the repository with all submodules:

```bash
git clone https://github.com/istio/istio.git
cd istio
git submodule update --init --recursive
```

The repository is structured as:
- `pilot/` - Service discovery and traffic management control plane
- `security/` - Certificate management and security policy
- `pkg/` - Shared utilities and Kubernetes integrations
- `tests/` - Integration tests
- `samples/` - Example applications

### Building the Control Plane

The Istio control plane is packaged as a single binary called `istiod` (which runs `pilot-discovery`). Build it with:

```bash
# Build the pilot-discovery binary (core control plane)
make pilot-discovery

# Or build all binaries (pilot-discovery, pilot-agent, istioctl, etc.)
make build
```

The compiled binary will be placed in `out/linux_amd64/` (or your architecture).

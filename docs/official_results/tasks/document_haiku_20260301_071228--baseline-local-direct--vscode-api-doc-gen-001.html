# VS Code Diagnostic API Reference

## Overview

The Diagnostic API allows VS Code extensions to report code issues (errors, warnings, information, or hints) to users within the editor. Diagnostics are displayed in the editor with visual indicators (squiggly lines, icons in the gutter) and in the Problems panel. The API provides a comprehensive system for managing diagnostic information across multiple documents and creating diagnostic collections.

The API is accessed through the `vscode.languages` namespace and provides tools for:
- Creating and managing diagnostic collections
- Publishing diagnostics for documents
- Listening to global diagnostic changes
- Providing contextual information about diagnostic issues

**Device**: Acer TravelMate 5735Z with Intel GM45/GMA 4500MHD GPU

**Data Structure**: `video_detect_dmi_table`  
**Confidence**: 0.95

### Required Fix

- DMI System Vendor: "Acer"
- DMI Product Name: "TravelMate 5735Z"

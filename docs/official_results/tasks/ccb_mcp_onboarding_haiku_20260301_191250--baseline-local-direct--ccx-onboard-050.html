## **Layer 2: API Server (kubernetes/kubernetes)**
The HTTP request is handled by **`createHandler()`** (handlers/create.go:53) which:
- Deserializes and validates the Deployment
- Calls **`REST.Create()`** via the REST storage interface

- **`Store.create()`** (store.go:477-545) that makes the critical call at line 526: **`e.Storage.Create()`** to persist to backend storage

- **Phase 2 (Commit):** Collects acknowledgments from all tasks, finalizes metadata when complete

**Five Major Components:**

1. **JobManager-Side (CheckpointCoordinator):**
   - Orchestrates checkpoint timing and task planning
   - Collects acknowledgments via `receiveAcknowledgeMessage()`
   - Aggregates state into `PendingCheckpoint`, converts to `CompletedCheckpoint`
   - Persists metadata to `CheckpointStorage`

2. **Barrier Injection & Propagation:**
   - `StreamTask` receives RPC trigger
   - Injects `CheckpointBarrier` via `OperatorChain.broadcastEvent()`
   - Barrier flows through data stream unmarked

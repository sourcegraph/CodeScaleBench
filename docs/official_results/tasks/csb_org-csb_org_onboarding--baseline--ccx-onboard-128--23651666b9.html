## Architecture Overview

The HotSpot JIT pipeline follows a tiered compilation strategy:
- **CompileBroker** orchestrates compilation requests and manages the workflow
- **CompilerThread** instances execute actual compilation tasks
- **AbstractCompiler** provides the common interface that both tiers inherit from
- **C1 Compiler** performs quick, first-tier compilation with moderate optimizations
- **C2 Compiler** performs aggressive second-tier optimization for hot methods

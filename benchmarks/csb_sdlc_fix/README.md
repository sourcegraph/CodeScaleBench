# csb_sdlc_fix

Bug fix tasks that require agents to identify and correct defects in large codebases, sourced from real-world issue reports.

## What This Suite Measures

Evaluates an agent's ability to reproduce bugs, locate faulty code, and apply correct fixes without introducing regressions. Tasks are derived from actual bug reports in open-source projects.

## Statistics

- **Task count:** 34
- **Difficulty:** 34 hard
- **SDLC phase:** fix

## Difficulty Rationale

Tasks are intentionally hard because they involve real bug reports from large repositories where the fix location is not obvious and requires cross-file analysis.

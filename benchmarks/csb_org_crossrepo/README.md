# csb_org_crossrepo

Cross-repository dependency analysis tasks that require agents to trace dependencies, APIs, and data flows across multiple repositories.

## What This Suite Measures

Evaluates an agent's ability to follow dependency chains, understand inter-repository API contracts, and analyze how changes in one repository affect others.

## Statistics

- **Task count:** 14
- **Difficulty:** 14 hard
- **SDLC phase:** org

## Difficulty Rationale

Tasks are intentionally hard because cross-repository analysis requires understanding build systems, package managers, and API boundaries across large codebases.

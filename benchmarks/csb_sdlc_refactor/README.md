# csb_sdlc_refactor

Refactoring tasks that require agents to restructure existing code while preserving behavior in large-scale projects.

## What This Suite Measures

Evaluates an agent's ability to improve code structure, reduce duplication, and apply design patterns without changing external behavior. Tasks require understanding dependencies and ensuring refactored code passes existing tests.

## Statistics

- **Task count:** 18
- **Difficulty:** 14 hard, 4 expert
- **SDLC phase:** refactor

## Difficulty Rationale

Tasks are intentionally hard/expert because refactoring at scale requires understanding the full impact of structural changes across large, interconnected codebases.

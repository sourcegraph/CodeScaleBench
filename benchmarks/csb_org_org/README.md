# csb_org_org

Organizational analysis tasks that require agents to understand and report on organization-wide code patterns, practices, and structures.

## What This Suite Measures

Evaluates an agent's ability to analyze an organization's full repository landscape, identifying patterns in code ownership, technology choices, and engineering practices.

## Statistics

- **Task count:** 16
- **Difficulty:** 15 hard, 1 medium
- **SDLC phase:** org

## Difficulty Rationale

Tasks are predominantly hard because organizational analysis requires broad understanding of multiple repositories and the ability to synthesize findings across an entire engineering organization.

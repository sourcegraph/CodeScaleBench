# csb_sdlc_secure

Security-focused tasks that require agents to identify vulnerabilities, apply security fixes, and implement secure coding practices.

## What This Suite Measures

Evaluates an agent's ability to find security flaws (e.g., injection, auth bypass, misconfiguration) and apply appropriate mitigations in large codebases. Tasks cover common vulnerability classes from the OWASP taxonomy.

## Statistics

- **Task count:** 15
- **Difficulty:** 15 hard
- **SDLC phase:** secure

## Difficulty Rationale

Tasks are intentionally hard because security analysis requires understanding both the application logic and the threat model, often spanning multiple components in large repositories.

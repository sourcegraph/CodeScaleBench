# csb_org_compliance

Compliance analysis tasks that require agents to audit codebases against organizational policies, standards, and regulatory requirements.

## What This Suite Measures

Evaluates an agent's ability to check code and configuration against compliance rules, identify violations, and report findings. Tasks involve multi-repository analysis using organizational context.

## Statistics

- **Task count:** 20
- **Difficulty:** 20 hard
- **SDLC phase:** org

## Difficulty Rationale

Tasks are intentionally hard because compliance analysis requires cross-repository reasoning, understanding organizational policies, and evaluating code against external standards.

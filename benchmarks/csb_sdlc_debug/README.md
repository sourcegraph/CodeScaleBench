# csb_sdlc_debug

Debugging tasks that require agents to diagnose and fix runtime errors, logic bugs, and unexpected behavior in large-scale codebases.

## What This Suite Measures

Evaluates an agent's ability to trace execution paths, interpret error messages, and identify root causes in complex software systems. Tasks span multiple languages and frameworks, requiring both local reasoning and cross-file analysis.

## Statistics

- **Task count:** 19
- **Difficulty:** 11 hard, 8 expert
- **SDLC phase:** debug

## Difficulty Rationale

Tasks are intentionally hard/expert because they target enterprise-scale debugging scenarios involving large repositories (often >1GB), multi-file root cause analysis, and non-trivial reproduction steps.

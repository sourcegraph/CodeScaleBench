# csb_org_onboarding

Developer onboarding tasks that require agents to orient new contributors by explaining codebase structure, conventions, and key components.

## What This Suite Measures

Evaluates an agent's ability to explore and explain unfamiliar organizational codebases, identifying entry points, architectural patterns, and developer workflows.

## Statistics

- **Task count:** 28
- **Difficulty:** 28 hard
- **SDLC phase:** org

## Difficulty Rationale

Tasks are intentionally hard because effective onboarding guidance requires comprehensive understanding of large, multi-repository organizational codebases and their conventions.

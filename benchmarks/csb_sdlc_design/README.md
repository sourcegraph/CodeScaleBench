# csb_sdlc_design

Design and architecture tasks that require agents to propose or evaluate software design decisions in large codebases.

## What This Suite Measures

Evaluates an agent's ability to reason about software architecture, API design, component interactions, and design trade-offs. Tasks require understanding existing system structure before proposing changes.

## Statistics

- **Task count:** 15
- **Difficulty:** 13 hard, 2 very_hard
- **SDLC phase:** design

## Difficulty Rationale

Tasks are intentionally hard because they require holistic understanding of large codebases and the ability to reason about design constraints, coupling, and extensibility at enterprise scale.

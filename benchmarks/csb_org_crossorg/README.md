# csb_org_crossorg

Cross-organization analysis tasks that require agents to reason about code patterns, dependencies, and practices across multiple organizations' repositories.

## What This Suite Measures

Evaluates an agent's ability to compare and analyze codebases from different organizations, identifying shared patterns, integration points, and organizational differences.

## Statistics

- **Task count:** 18
- **Difficulty:** 18 hard
- **SDLC phase:** org

## Difficulty Rationale

Tasks are intentionally hard because cross-organization analysis requires reasoning across multiple large repositories with different conventions and architectures.

# csb_sdlc_feature

Feature implementation tasks that require agents to add new functionality to existing large-scale codebases.

## What This Suite Measures

Evaluates an agent's ability to understand existing code structure, identify integration points, and implement new features that conform to project conventions. Tasks involve real-world feature requests from open-source projects.

## Statistics

- **Task count:** 23
- **Difficulty:** 14 hard, 9 expert
- **SDLC phase:** feature

## Difficulty Rationale

Tasks are intentionally hard/expert because they require navigating large repositories, understanding existing patterns, and producing code that integrates correctly with complex systems.

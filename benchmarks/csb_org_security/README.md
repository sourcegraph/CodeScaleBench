# csb_org_security

Organization-wide security analysis tasks that require agents to assess security posture across multiple repositories.

## What This Suite Measures

Evaluates an agent's ability to identify security vulnerabilities, misconfigurations, and policy violations across an organization's codebase, including dependency analysis and access control review.

## Statistics

- **Task count:** 26
- **Difficulty:** 26 hard
- **SDLC phase:** org

## Difficulty Rationale

Tasks are intentionally hard because organization-wide security analysis requires understanding threat models, scanning multiple repositories, and correlating findings across services.

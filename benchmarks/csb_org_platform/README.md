# csb_org_platform

Platform engineering tasks that require agents to analyze shared infrastructure, internal tools, and platform services across an organization.

## What This Suite Measures

Evaluates an agent's ability to understand platform abstractions, shared libraries, CI/CD pipelines, and infrastructure-as-code across organizational repositories.

## Statistics

- **Task count:** 20
- **Difficulty:** 20 hard
- **SDLC phase:** org

## Difficulty Rationale

Tasks are intentionally hard because platform analysis requires understanding infrastructure patterns, deployment configurations, and shared tooling across multiple repositories.

# csb_org_incident

Incident investigation tasks that require agents to diagnose production issues by analyzing code, logs, and system behavior across an organization's repositories.

## What This Suite Measures

Evaluates an agent's ability to perform root cause analysis for production incidents, tracing issues through multiple services and identifying the source of failures.

## Statistics

- **Task count:** 20
- **Difficulty:** 20 hard
- **SDLC phase:** org

## Difficulty Rationale

Tasks are intentionally hard because incident investigation requires correlating evidence across multiple repositories, understanding system interactions, and reasoning about failure modes.

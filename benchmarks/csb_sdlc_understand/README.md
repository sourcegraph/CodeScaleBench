# csb_sdlc_understand

Code comprehension tasks that require agents to answer questions about code structure, behavior, and dependencies.

## What This Suite Measures

Evaluates an agent's ability to read and reason about unfamiliar large-scale codebases, answering questions about control flow, data flow, component relationships, and system behavior.

## Statistics

- **Task count:** 12
- **Difficulty:** 12 hard
- **SDLC phase:** understand

## Difficulty Rationale

Tasks are intentionally hard because they require synthesizing information from multiple files across large repositories to answer non-trivial questions about system behavior.

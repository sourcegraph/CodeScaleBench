# csb_org_crossrepo_tracing

Cross-repository tracing tasks that require agents to follow execution paths, configuration flows, and dependency chains across repository boundaries.

## What This Suite Measures

Evaluates an agent's ability to trace how code, configuration, and data flow across multiple repositories in an organization, identifying the full path from source to effect.

## Statistics

- **Task count:** 23
- **Difficulty:** 23 hard
- **SDLC phase:** org

## Difficulty Rationale

Tasks are intentionally hard because tracing across repositories requires deep understanding of inter-service communication, shared libraries, and configuration propagation.

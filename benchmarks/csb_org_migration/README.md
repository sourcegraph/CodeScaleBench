# csb_org_migration

Migration planning tasks that require agents to analyze codebases and plan migrations across libraries, frameworks, or infrastructure.

## What This Suite Measures

Evaluates an agent's ability to assess migration scope, identify breaking changes, and plan migration strategies across an organization's repositories.

## Statistics

- **Task count:** 28
- **Difficulty:** 28 hard
- **SDLC phase:** org

## Difficulty Rationale

Tasks are intentionally hard because migration analysis requires understanding version compatibility, API changes, and the full dependency graph across large, multi-repository codebases.

# csb_sdlc_test

Test authoring tasks that require agents to write or improve test suites for large software projects.

## What This Suite Measures

Evaluates an agent's ability to identify untested code paths, write meaningful test cases, and achieve adequate coverage. Tasks require understanding both the code under test and the project's testing conventions.

## Statistics

- **Task count:** 20
- **Difficulty:** 18 hard, 2 medium
- **SDLC phase:** test

## Difficulty Rationale

Tasks are predominantly hard because writing effective tests for large codebases requires understanding complex interactions, edge cases, and the project's testing infrastructure.

# csb_sdlc_document

Documentation tasks that require agents to generate, update, or evaluate technical documentation for large software projects.

## What This Suite Measures

Evaluates an agent's ability to understand code behavior and produce accurate, complete documentation including API references, usage guides, and inline comments. Tasks require reading and synthesizing information across multiple files.

## Statistics

- **Task count:** 15
- **Difficulty:** 15 hard
- **SDLC phase:** document

## Difficulty Rationale

Tasks are intentionally hard because they require deep comprehension of large codebases and the ability to accurately describe complex system behavior and interfaces.

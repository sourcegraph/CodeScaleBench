# csb_org_domain

Domain knowledge tasks that require agents to apply domain-specific expertise when analyzing organizational codebases.

## What This Suite Measures

Evaluates an agent's ability to understand and reason about domain-specific code patterns, business logic, and specialized frameworks within an organization's repositories.

## Statistics

- **Task count:** 20
- **Difficulty:** 20 hard
- **SDLC phase:** org

## Difficulty Rationale

Tasks are intentionally hard because they require both technical code understanding and domain-specific knowledge to correctly analyze organizational codebases.

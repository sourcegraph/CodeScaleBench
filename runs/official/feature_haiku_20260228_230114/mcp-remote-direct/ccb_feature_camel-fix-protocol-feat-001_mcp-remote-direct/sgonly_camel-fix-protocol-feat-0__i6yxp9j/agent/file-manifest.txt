INTEGRATION POINTS
==============================================================================

1. Component Discovery
   → META-INF/services file enables auto-discovery
   → Camel will load via ServiceLoader at startup

2. URI Registration
   → @UriEndpoint registers scheme "fix"
   → @UriPath registers sessionID parameter
   → @UriParam registers query parameters

3. Configuration Injection
   → setProperties() from DefaultComponent
   → Maps URI parameters to POJO setters
   → Enables: fix:SESSION1?senderCompID=TRADER

4. Producer/Consumer Creation
   → createProducer() → FixProducer
   → createConsumer() → FixConsumer
   → Both receive FixEndpoint reference

5. Async Support
   → DefaultAsyncProducer handles threading
   → AsyncCallback enables non-blocking sends
   → Compatible with Camel's async routing

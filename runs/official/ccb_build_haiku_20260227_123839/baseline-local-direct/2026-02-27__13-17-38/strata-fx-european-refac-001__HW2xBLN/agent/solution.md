# FxVanillaOption to FxEuropeanOption Refactoring - Complete Analysis

## Executive Summary

This document provides a comprehensive refactoring plan to rename `FxVanillaOption` to `FxEuropeanOption` throughout the OpenGamma Strata codebase. The refactoring involves 45 files across four major modules (product, pricer, measure, loader), affecting ~500 class name references, imports, and method calls.

    local old_path="$1"
    local new_path="$2"
    
    if [ -f "$old_path" ]; then
        # Get class names from filenames
        local old_class=$(basename "$old_path" .java)
        local new_class=$(basename "$new_path" .java)
        
        # Create temp file with replacements
        sed "s/$old_class/$new_class/g" "$old_path" > "$old_path.tmp"
        
        # Move to new location
        mv "$old_path.tmp" "$new_path"
        
        echo "Renamed: $old_path -> $new_path"

QUICK START
───────────

1. Start with README.md (5-10 min read)

2. Copy all code from complete_implementation.md to:
   - modules/product/src/main/java/com/opengamma/strata/product/credit/
   - modules/pricer/src/main/java/com/opengamma/strata/pricer/credit/

4. Compile: mvn clean compile -pl product,pricer,measure

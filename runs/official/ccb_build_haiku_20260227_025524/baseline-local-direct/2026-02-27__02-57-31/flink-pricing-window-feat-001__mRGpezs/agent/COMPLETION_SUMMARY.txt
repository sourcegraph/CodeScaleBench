================================================================================
USAGE EXAMPLES
================================================================================

Example 1: NYSE Trading Sessions
```java
PricingSessionWindow nyseWindows = PricingSessionWindow.forMarket(
    "NYSE",
    ZoneId.of("America/New_York"),
    LocalTime.of(9, 30),
    LocalTime.of(16, 0)
);

DataStream<Trade> windowed = trades.keyBy(t -> t.getSymbol())
    .window(nyseWindows)
    .apply(...);  // Fires at 16:00 ET (market close)
```

Example 2: Add Custom Market
```java
PricingSessionWindow.registerMarket(
    "EUREX",
    ZoneId.of("Europe/Berlin"),
    LocalTime.of(8, 0),
    LocalTime.of(22, 0),
    false
);
```

Example 3: CME Futures (Overnight Session)
```java
PricingSessionWindow cmeWindows = PricingSessionWindow.forMarket(
    "CME_ES",
    ZoneId.of("America/Chicago"),
    LocalTime.of(17, 0),
    LocalTime.of(16, 0),
    true  // isOvernightSession
);
```
